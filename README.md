# code-quiz

## Description
This project is a coding quiz that will start when the start button is clicked. Once the quiz has started the user is presented with a question and a timer will start. Once all the questions are answered or time reaches zero, the quiz will end and a user will be able to view and save their high score.Built

## Built with
* JavaScript
* HTML
* CSS
